A Demo Social Authentication Application using Okta's APIs.

## Technologies used:

### Front-end

1. jQuery
2. Jade
3. Foundation

### Back-end

1. Okta's APIs
2. NodeJS
3. Web Storage

## Installing (WIP)

1. Get the packages specified in packages.json installed 
2. Run  with ```npm start```
3. Open the server at ```http://localhost:5000```
